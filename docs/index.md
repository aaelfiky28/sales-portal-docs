## Quick Start

Documentation is automatically generated when you create a pull request with a Jira ticket.

### For Developers
1. Create PR with Jira ticket in title (e.g., `SP-123: Add feature`)
2. Documentation PR is automatically created
3. Review and enhance the AI-generated content
4. Merge documentation PR when feature is complete

### For Product Managers
- View business documentation in the [Business section](/business/)
- Track feature progress through Jira integration
- Review user stories and acceptance criteria
